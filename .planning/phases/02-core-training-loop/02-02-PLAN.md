<objective>
Build the complete training plan builder for the trainer: a form-based interface with nested dynamic fields for training days and exercises, exercise picker from the existing library, cycle length configuration, and plan versioning (create draft from active, activate new version). The trainer can create, view, and edit plans for any client.

Purpose: The plan builder is the trainer's primary tool for prescribing workouts. Javier needs to create structured training plans (organized by days with specific exercises, sets, reps, and weights) for each of his 20-50 clients. Plan versioning ensures historical workout data is preserved when plans change.

Output: Plan builder form, exercise picker dialog, plan view page, plan edit page, versioning workflow, all connected to routes in App.tsx.
</objective>
