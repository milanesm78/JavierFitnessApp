Wrap with a "kg" suffix label.

    - Update the `onSubmit` call: `onSubmit({ name: name.trim(), youtube_url: youtubeUrl.trim(), description: description.trim(), default_weight_kg: defaultWeight.trim() })`

    **useExercises hook (src/features/exercises/hooks/useExercises.ts):**
    No changes needed to the hook file itself. The `ExerciseFormData` type flows through and Supabase accepts the new fields. However, the `useCreateExercise` and `useUpdateExercise` mutations pass `ExerciseFormData` directly to `.insert()` and `.update()`. Since `default_weight_kg` is a string in the form but numeric in the DB, add a transformation in each mutation's `mutationFn`:
    ```
    const payload = {
      name: exercise.name,
      youtube_url: exercise.youtube_url,
      description: exercise.description || null,
      default_weight_kg: exercise.default_weight_kg ? parseFloat(exercise.default_weight_kg.replace(',', '.')) : null,
    };
    ```
    Use `parseFloat` with comma-to-period normalization (consistent with DecimalInput's normalizeDecimal pattern). Pass `payload` instead of `exercise` to `.insert()` and `.update()`.
