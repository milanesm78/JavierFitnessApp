    2. Create `src/features/exercises/hooks/useExercises.ts`:
       - Use query key factory pattern: `exerciseKeys = { all: ['exercises'], list: () => [...all, 'list'], detail: (id) => [...all, 'detail', id] }`
       - `useExercises()`: TanStack `useQuery` fetching `supabase.from('exercises').select('*').order('name')`. Return typed `Exercise[]`.
       - `useCreateExercise()`: TanStack `useMutation` calling `supabase.from('exercises').insert(exercise).select().single()`. On success: `invalidateQueries({ queryKey: exerciseKeys.list() })`. Show success toast.
       - `useUpdateExercise()`: TanStack `useMutation` calling `supabase.from('exercises').update(updates).eq('id', id).select().single()`. On success: invalidate list + detail queries. Show success toast.
       - `useDeleteExercise()`: TanStack `useMutation` calling `supabase.from('exercises').delete().eq('id', id)`. On success: invalidate list. Show success toast.
       - All mutations should show error toasts on failure.
       - See RESEARCH.md "Pattern 3: TanStack Query with Supabase"
